The following resources have all been invaluable in getting this site up and running. Note that all of the sites listed here provide their information and services for free.

Obey the Testing Goat!
I started out creating my juggling site by following the examples in Harry Percival's excellent book "Test-Driven Development with Python". I'd recommend buying a print copy if you're interested in web development with Python.
Visit the Obey the Testing Goat! site to see the free online version of the book.
Django
This site was built using the Django framework.
Head over to the Django project website to find out more.
Python
The Django framework is based on Python (the best programming language out there, IMO).
Visit Python website to download Python or find out more.
The Mozilla Development Network (MDN)
Courses covering various web development topics, including HTML, CSS & JavaScript.
Check out MDN's Learn Web Development page, especially if you're interested in learning front-end web development.
