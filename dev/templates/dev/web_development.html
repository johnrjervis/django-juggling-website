Overview

After teaching myself Python programming, I decided to build a website to showcase my juggling skills.

To achieve this, I learnt HTML, CSS and JavaScript from the Mozilla Developer Network's (MDN) web developer courses and launched a front-end version of my website on GitHub pages. I then learnt Django and built a full-stack version of my website by adapting the example project from Harry Percival's book Test-Driven Development with Python.

Skills and experience

While creating my website, I gained experience of the following technologies (where possible, links to examples in my GitHub repositories have been provided):

  • Functional testing with Selenium: I wrote functional tests with the Selenium package for Python to simulate user interactions on all the pages of my website.
  • Unit testing: I created unit tests with Django's test module to test the views, models and forms of my website.
  • Web forms: Site visitors can post comments on each video via forms on my website. I created these using Django's ModelForm class.
  • Responsive design: I used media queries so that my website adapts its style to the size of the browser window that it is displayed in.
  • Database management: I used SQLite database tables to store user comments and links to video files. The two tables are linked by a foreign key relationship.
  • Django's admin site: I created an administration page for my website so that I can easily add new videos and moderate visitor comments.
  • Deployment: I deployed my website through PythonAnywhere: this included editing the WSGI file, adjusting the security settings in Django, and setting environmental variables.
