Overview

After teaching myself Python programming, I decided to build a website to showcase my juggling skills (see www.johnrjervis.co.uk).

To achieve this, I learnt HTML, CSS and JavaScript from the Mozilla Developer Network's (MDN) web developer courses and launched a front-end version of my website on GitHub pages. I then taught myself Django and built the full-stack version of my website by adapting the example project from Harry Percival's book Test-Driven Development with Python.

Having gained plenty of experience of the basics of front-end development through working on my website, I have now started to learn some more advanced technologies, such as jQuery and React.

Skills and experience

I have experience of the following technologies (where possible, links to examples on GitHub or CodePen have been provided):

  • Functional testing with Selenium: I wrote functional tests with the Selenium package for Python to simulate user interactions on all the pages of my website.
  • Unit testing in Python: I created unit tests with Django's test module to test the views, models and forms of my site.
  • Unit testing in JavaScript: I used QUnit to test my website's JavaScript code.
  • DOM manipulation with jQuery: I wrote event handlers with jQuery to provide custom error messages for form validation and a flyout menu on my site.
  • Object-oriented programming with JavaScript: I created a concept for a juggling animation with the Canvas API (view the animation live via this link).
  • Web forms: Site visitors can post comments on each video via forms on my website. I created these using Django's ModelForm class.
  • Responsive design: I used media queries so that my website adapts itself to the size of the browser window that it is displayed in.
  • Database management: I used SQLite database tables to store user comments and links to video files for my website. The two tables are linked by a foreign key relationship.
  • Django's admin site: I created an administration page for my website so that I can easily add new videos and moderate visitor comments.
  • Deployment: I deployed my site through PythonAnywhere.
  • React: I have started learning to use the React library through freeCodeCamp, and have built a random quote generator on CodePen.
